Back and Next

Instead of a navigation bar with the “Back” and “Forward” buttons of a standard browser like Internet Explorer, some of MyAccount courses have "Back” and “Next” navigation buttons (shown as < and >) located at the top and bottom of the right hand side of the screen. Selecting the “<” (or Back) button will take you to the previously viewed page and selecting the “>” (or Next) button will bring you to the next page.

“Français” – French Toggle

Select “Français” to switch from the English version of the course to the French version.

Course Home

Select the Home icon to return to the course Home page.

Help

Select the Help icon to access these help topics again.

Toolbox

Select the Toolbox icon to access the course glossary and links to useful tools, guidelines, and other information related to this course.

Sitemap

Select the Sitemap icon to display an exhaustive list of all pages and sections of the course.

Exit

Select the Exit icon when you want to exit this course. A bookmark is automatically created to mark your current location in the course. The next time you launch the course you will have return to where you left off.
