Using Components

There are several different components available for formatting your content. The Snap! Framework is built on the WET template and utilizes the widgets that have been created at part of that template. To align the look and feel of courses built using the Snap! framework with those built in Adapt, we will be using a 2-column layout. Components can either be full-width or half-width of the block. Within each block the maximum number of components is limited to 2. This allows the using to focus on one block of content at a time, similar to a slide in Captivate.

Icon Boxes

Use Icons Boxes in conjunction with the Common Look and Feel icons. Each standard icon has an associated use and is outlined in the Style Guide section entitled Colours and Icons. Icon boxes are styled differently than regular text components to draw the learner's focus to special information.
